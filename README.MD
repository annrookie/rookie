#Rookie-common  
　　*通用包工具，主要包含了多种简单的工具类。  
　　* 结构目录如下：  
　　　　|--com  
　　　　　　|--rookie  
　　　　　　　　|--common  
　　　　　　　　　　|--exception 异常处理  
　　　　　　　　　　|--filter    过滤器接口  
　　　　　　　　　　|--resource  静态常量类资源  
　　　　　　　　　　|--util      工具类  
　　* 工具类：  
　　　1.StrUtil：字符串工具类  
　　　2.ArrayUtil：数组工具类  
　　　3.ComUtil：校验工具类  
　　　4.IdcardUtil：身份证工具类  
　　　5.DateUtil：日期工具类【开发】
