# rookie  
* 项目说明  
```
此项目主要致力于JAVA工具类开发，主要针对常用的方法进行封装。  
由于项目作者水平有限，如若方法有误或是有更好的优化方法，欢迎指出，谢谢。
```

** 项目结构：
```
rookie-common

      ├── rookie
            ├── common
```
